### Personal portfolio website built with GatsbyJS & TypeScript

### Live URL: https://mohib.me

### Install & Run

1.  First clone the repo:

```bash
$ git clone git@github.com:mbmohib/mohib.git
```

2.  Then install the dependencies:

```bash
$ npm install
```

3.  Finally run the app:

```bash
$ npm start
```
